# Practical 2

## Requirements

- Python 3.11

## Installation

- create a virtual environment

```bash
python -m venv .venv
source .venv/bin/activate
```

- install the requirements

```bash
pip install -r requirements.txt
```

- download the dataset

```bash
python scripts/download_dataset.py
```

- run the experiments

```bash
python src/main.py
```
